# S/4HANA RAP Purchase Order Approval System

## Overview
RAP-based Purchase Order Management with:
- Vendor validation
- Approval actions (Approve / Reject)
- OData V4 API for Fiori consumption

## Features
- Create / Update / Delete Purchase Orders
- Approve or Reject Purchase Orders
- Real-time vendor validation
- Fiori-ready CDS views

## Tech Stack
- S/4HANA, ABAP RAP, CDS Views
- Behavior Definition + Actions
- OData V4 Service

## How to Use
1. Import all objects into ADT (Eclipse)
2. Activate in order: CDS → Behavior → Service Definition → Service Binding
3. Test via Fiori OData Service
